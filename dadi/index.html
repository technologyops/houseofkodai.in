monitor data that's critical to you

DaDi (Data Display) enables managers to make better decisions, by monitoring mission critical information in real-time. It achieves this by collating real time information from various sources of the organization and pushes it immediately to one or more desktops.

Designed on the concept similar to Television Broadcast, it provides the flexibility of choosing what information needs to be displayed and also provides the necessary security features for restricted access.

DaDi works in existing corporate networks. It can work via. the Intranet/Internet and support roaming users via. VPN. It integrates seamlessly via. firewalls/proxys and intrusion detection systems, requiring no changes to existing network infrastructure. IT administrators find it a joy to work with; deploying the client in desktops takes seconds. Since it works with existing web-servers, it requires no complicated setups. Application Developers who would write custom data adapters for data channels will find our API simple and easy to program with.

DaDi is indispensable for managers who want to:
  • Monitor mission critical information in real-time
  • Get notified of critical events, enabling them to take corrective action
  • Track progress of specific tasks
  • Broadcast news/event information to all/selected desktops
